Simple URL Shortener

About

This project is a simple URL shortener web application developed by Krishna Keshri. It allows users to input long URLs and generate shortened versions that are easier to share and remember. The app features a clean and user-friendly interface, making URL shortening quick and efficient. It is built using HTML, CSS, and JavaScript, and demonstrates fundamental web development concepts such as form handling, DOM manipulation, and event-driven programming.
